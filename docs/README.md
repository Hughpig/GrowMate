## 🚀 项目概览

GrowMate 是一个陪伴成长型综合社区平台，核心特性包括：

### 🎯 核心功能
- **学习成长系统**：四大专业模块（技术、体能、营养、心理）
- **AI 智能功能**：个人档案、陪伴聊天、智能推荐
- **社区互动**：垂直社区、成长日记、情绪打卡

### 🛠️ 技术栈
- **前端**：Next.js 15 + TypeScript + Tailwind CSS
- **后端**：Node.js + Prisma + SQLite/PostgreSQL
- **AI**：OpenAI API + 自然语言处理

### 📱 访问方式
- **开发环境**：http://localhost:3000
- **测试账号**：demo@growmate.app / demo123456
