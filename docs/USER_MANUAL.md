# GrowMate 用户手册 & 测试指南

## 📖 项目概述

GrowMate 是一个陪伴成长型综合社区平台，集成了 AI 个人档案、多领域成长赋能和垂直陪伴社区功能。本手册将帮助你快速上手和全面测试平台功能。

---

## 🚀 快速开始

### 环境要求
- Node.js >= 18.0
- npm >= 8.0
- SQLite (开发环境)

### 安装运行
```bash
# 1. 克隆项目
git clone https://github.com/JenniferJJiang/GrowMate.git
cd GrowMate

# 2. 安装依赖
npm install

# 3. 设置环境
cp apps/web/.env.example apps/web/.env
# 编辑 .env 文件，保持默认配置即可

# 4. 初始化数据库
npm run db:generate
npm run db:push
npm run db:seed

# 5. 启动应用
npm run dev
```

**访问地址**：http://localhost:3000

## 👤 用户功能指南

### 1. 登录注册
- 访问首页，点击"开启档案"进行注册
- 或使用演示账号直接登录体验
- 支持邮箱密码登录

### 2. 仪表板 (/dashboard)
- **总览**：查看学习进度、社区动态、AI 互动统计
- **快捷入口**：直接访问各个功能模块
- **个人信息**：查看和编辑个人资料

### 3. 学习模块 (/learn)
#### 核心功能
- **四大课程模块**：技术学习、体能训练、营养搭配、心理健康
- **进度追踪**：可视化进度条和完成统计
- **课程筛选**：按级别、时长搜索课程
- **学习工具**：计时器、笔记、收藏功能

#### 使用流程
1. 选择学习模块（如"技术学习"）
2. 浏览课程列表，使用搜索筛选功能
3. 点击课程查看详细内容
4. 使用"开始学习计时"记录学习时间
5. 添加学习笔记和心得
6. 完成课程后点击"标记完成"

#### 特色功能
- **内容预览**：长内容支持展开/收起
- **学习建议**：基于进度的个性化推荐
- **成就系统**：学习徽章和进度激励
- **数据分析**：学习时长和课程分布统计

### 4. 成长日记 (/journal)
- **记录感悟**：编写成长心得和反思
- **分享交流**：发布日记到社区
- **隐私设置**：公开或私密日记选项

### 5. 情绪打卡 (/mood)
- **情绪记录**：每日心情评分（1-5分）
- **趋势分析**：情绪变化可视化
- **建议推荐**：基于情绪状态的建议

### 6. AI 档案 (/archive)
- **智能分析**：AI 分析成长数据
- **档案生成**：自动生成个人成长报告
- **趋势预测**：基于历史数据的发展预测

### 7. AI 陪伴 (/ai)
- **对话聊天**：与 AI 伴侣进行自然对话
- **情绪支持**：24/7 情绪疏导和鼓励
- **学习指导**：AI 提供学习建议和答疑

### 8. 社区 (/community)
- **四大社区**：成长日记、情绪心理、运动体能、技术学习
- **发帖互动**：发布话题、回复、点赞
- **兴趣小组**：基于兴趣的主题讨论

## 🛠️ 开发者指南

### 项目结构
```
apps/web/src/
├── app/                    # Next.js App Router
│   ├── (main)/           # 主要页面
│   │   ├── dashboard/    # 仪表板
│   │   ├── learn/        # 学习模块
│   │   └── ...
│   ├── components/       # React 组件
│   ├── lib/             # 工具库
│   └── types/           # TypeScript 类型
├── prisma/              # 数据库 schema
└── public/             # 静态资源
```

### 开发命令
```bash
# 开发环境
npm run dev              # 启动开发服务器
npm run db:generate      # 生成 Prisma 客户端
npm run db:push         # 推送数据库 schema
npm run db:seed         # 填充测试数据

# 生产环境
npm run build           # 构建生产版本
npm run start           # 启动生产服务器
```

### 数据库操作
```bash
# 查看数据库
npm run db:studio       # 打开 Prisma Studio

# 重置数据库
rm -rf apps/web/dev.db
npm run db:push
npm run db:seed
```

### 添加新功能
1. **数据库设计**：更新 `prisma/schema.prisma`
2. **API 接口**：在 `src/app/api/` 创建路由
3. **前端组件**：在 `src/components/` 创建组件
4. **页面路由**：在 `src/app/` 创建页面
5. **类型定义**：在 `src/types/` 添加类型

## 🔧 故障排除

### 常见问题

**Q: 应用无法启动？**
```bash
# 检查端口占用
lsof -i :3000

# 清理缓存
rm -rf apps/web/.next
npm run dev
```

**Q: 数据库连接失败？**
```bash
# 重新初始化数据库
rm -rf apps/web/dev.db
npm run db:generate
npm run db:push
npm run db:seed
```

**Q: 样式不显示？**
```bash
# 重启开发服务器
npm run dev
```

### 调试技巧
```bash
# 查看详细日志
npm run dev -- --verbose

# 开启 TypeScript 严格模式
npm run dev -- --tsc
```
