# Psychological Mapping Reference

Use this reference only to guide lightweight text interpretation. Do not turn these frameworks into clinical diagnosis or deterministic personality labels.

## Aaker Brand Personality Five Dimensions

Aaker's five dimensions can be used as a language for perceived personality style:

| Dimension | Chinese Label | Common Signals |
|---|---|---|
| Sincerity | 真诚 | 关心、陪伴、温暖、踏实、家庭、朋友、可信 |
| Excitement | 兴奋 | 新鲜、冒险、潮流、探索、热闹、变化、创造 |
| Competence | 能力 | 工作、效率、目标、专业、可靠、解决问题、掌控 |
| Sophistication | 精致 | 审美、穿搭、品味、优雅、品质、社交形象 |
| Ruggedness | 坚韧 | 运动、户外、抗压、独立、强硬、耐力 |

When mapping a person, phrase the result as “呈现出类似 X 的表达风格” or “文本更接近 X 线索”, not as a fixed identity.

## Schwartz Basic Values Ten Dimensions

Schwartz's ten values can be used to infer motivational priorities:

| Value | Chinese Label | Common Signals |
|---|---|---|
| Self-Direction | 自主 | 自由选择、独立决定、创造、探索自我 |
| Stimulation | 刺激 | 新鲜感、变化、挑战、冒险 |
| Hedonism | 享乐 | 快乐、吃喝、休闲、舒适、即时满足 |
| Achievement | 成就 | 目标、优秀、竞争、认可、完成任务 |
| Power | 权力 | 影响力、控制、地位、资源、主导 |
| Security | 安全 | 稳定、健康、秩序、风险规避、保障 |
| Conformity | 遵从 | 规则、克制、不给别人添麻烦、按要求做 |
| Tradition | 传统 | 习俗、长辈、仪式、延续、文化身份 |
| Benevolence | 仁爱 | 朋友、家人、照顾、陪伴、可靠关系 |
| Universalism | 普世关怀 | 公平、环保、社会议题、包容、公共利益 |

Prefer one or two strongest value hypotheses. If no value is clear, use “无明确价值观倾向”.

## Emotion And Happiness Heuristics

Emotion:

- Stable: calm, routine, no strong conflict, low volatility.
- Positive: joy, gratitude, connection, relief, satisfaction.
- Negative: sadness, anger, anxiety, fatigue, frustration.
- Mixed: both reward and conflict are present.
- Unclear: sentence lacks emotional words or context.

Happiness:

- High: positive emotion plus connection, achievement, rest, meaning, or enjoyment.
- Medium: neutral or mildly positive routine without strong distress.
- Low: clear distress, loneliness, failure, conflict, exhaustion, or helplessness.
- Neutral: insufficient evidence.

## Confidence Rules

- High: the sentence explicitly states the cue.
- Medium: the cue is strongly implied by activity and tone.
- Low: the cue is possible but weak or context-dependent.
